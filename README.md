# AI Expense Management Dashboard

## Overview

AI Expense Management Dashboard is a simple frontend application that simulates an AI-powered expense approval workflow.

The dashboard allows users to submit expenses and automatically determines whether an expense should be auto-approved or sent for manual review based on a predefined threshold.

## Features

* Submit expense details
* Automatic approval for expenses below $100
* Human review required for expenses above $100
* Real-time dashboard statistics
* Simple and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Workflow

1. User enters expense details.
2. System checks the expense amount.
3. If amount is below $100:

   * Auto Approved
4. If amount is $100 or above:

   * Human Review Required
5. Dashboard statistics are updated automatically.
